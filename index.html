Disclaimer: This is an entirely unofficial tool, completely unaffiliated with the IAA. It's provided as is, may have bugs and rather than using the JSON provided by the IAA uses JSON that has been reverse engineered from the KMZ file provided by the IAA. See the git commit history for more on this.

The map interface has two settings on the left hand side, under the +/- zoom buttons. The round circle icon will allow you set the operational zone to the radius defined in the form when you click the map and will set the latitude and longitude of the location in the form. The marker icon will allow you set the operator location. The map will attempt to determine the controlled airspace from the layer clicked. This has not been extensively tested. The name of the location of the operator is required after the name of the airspace zone. Eg. "Tallaght" or "5 miles northwest of Rathcoole". The Layer for the controlled airspace will not appear on the map in the generated UF101 form. This is by design.
